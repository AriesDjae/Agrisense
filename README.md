# Agrisense
Agrisense adalah web yang dibuat dengan tujuan membantu petani Indonesia memilih tanaman apa yang cocok untuk ditanam berdasarkan kondisi tanah dan cuaca.

# Cara Kerja
Petani memasukkan wilayah tempat tinggal petani. Website tersebut lalu memberikan rekomendasi tanaman yang cocok untuk ditanam berdasarkan cuaca wilayah dan kondisi tanah.
